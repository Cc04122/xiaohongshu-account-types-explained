## 常见账号类型

目前市场上主要有：

- 白号（新号）  
- 满月号  
- 实名号  
- 老号  
- 海外账号  

---

## 不同账号的区别

### 1️⃣ 白号

- 无历史  
- 权重低  
- 起号难  

👉 适合测试，不适合变现  

---

### 2️⃣ 满月号

- 已过沙盒期  
- 初始权重较好  

👉 适合入门  

---

### 3️⃣ 实名号

- 稳定性更高  
- 风控更低  

👉 适合长期运营  

---

### 4️⃣ 老号

- 有历史行为  
- 更容易起量  

👉 适合引流 / 带货  

---

## 为什么账号类型很重要？

因为小红书的推荐机制，会参考：

- 历史行为  
- 账号标签  
- 风控评分  

👉 不同账号，起点完全不同  

---

## 如何选择？

如果你是：

- 新手 → 满月号  
- 做变现 → 实名号 / 老号  
- 做矩阵 → 批量账号
